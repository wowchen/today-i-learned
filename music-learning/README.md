# 音乐学习站 · Music Learning

用大白话讲透音乐的零基础自学站:**乐理 / 乐器 / 音乐史 / 赏析 / 流行歌演唱**。
从认识一个音符,到把一首流行歌唱出来。纯静态、零依赖、离线可用,术语点哪查哪。

是「今日得到 / Today I Learned」合集中的一个站,视觉定稿为**风格 A 黑胶夜场**。

## 怎么用

- 顶部导航:**首页 / 学习路线 / 术语表 / 术语本 / 搜索 / 设置**。
- 跟着「学习路线」一节节往下学;只想学唱歌可直奔模块 5「流行演唱」。
- 正文里带虚线的词点开即可看大白话解释,可收藏进「术语本」翻牌复习。
- 进度自动存在浏览器本机;在「设置」里可选配 GitHub 私有仓库同步。

## 本地运行

```bash
# 在仓库根目录
python3 -m http.server 8011
# 浏览器打开 http://localhost:8011/music-learning/
```
直接双击 `index.html`(`file://`)也能用。

## 目录一览

```
index.html        入口(hash 路由单页)
design/           风格选型稿(A/B/C),定稿 A,保留
assets/css/       全站样式(风格 A 黑胶夜场)
assets/js/        SPA 引擎(命名空间 MUS,勿改)
content/          模块、术语、各模块课程文件
```

开发约定与"新增一课三步"见 `CLAUDE.md`,定位与决策见 `需求与讨论.md`。
