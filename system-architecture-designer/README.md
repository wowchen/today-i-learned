# 架构学习站 · 系统架构设计师(软考高级)

用大白话讲透**系统架构设计师**考点的备考站。三科全覆盖:综合知识(选择)、案例分析、论文。
纯静态、零依赖、离线可用;真题练习即时判分、错题本、架构计算器、术语点查。

是「今日得到 / Today I Learned」合集中的一个站,引擎沿用高项站(瑞士编辑型 + 蓝图钢青配色与架构元素)。

## 怎么用

- 顶部导航:**课时 / 术语 / 收藏 / 错题 / 计算器 / 搜索 / 设置**
- 跟着首页的学习路线走(基础→核心→进阶→实战);也可点模块卡片跳学
- 每课⑤有真题练习,答错自动进错题本;正文带下划线的术语点开即看、可收藏
- 计算器内置:系统可靠度(串/并联)、Amdahl 加速比、海明校验位
- 进度/收藏/错题存在浏览器本地,可在设置导出备份

## 本地运行

```bash
# 仓库根目录
python3 -m http.server 8011
# 打开 http://localhost:8011/system-architecture-designer/
```

## 内容

20 个模块、133 节五分钟微课:
- 基础:计算机系统 / 操作系统 / 数据库 / 网络
- 核心:软件工程 / 设计模式 / 信息安全与安全架构 / 软件架构基础 / 架构风格 / 质量属性与架构评估 / 软件可靠性
- 进阶:系统性能 / 中间件·SOA·微服务 / 嵌入式 / 架构演化 / 未来技术 / 标准化与知识产权
- 实战:案例分析专题 / 论文写作专题

> 考试题量/分值/法律年限等以官方最新通知与现行法律为准。开发约定见 `CLAUDE.md`。
