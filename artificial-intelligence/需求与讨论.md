# 需求与讨论 · 人工智能学习站

记录本站从立项到落地的关键决策,便于回溯。

## 起点

用户:"新建站 人工智能"。定位为「今日得到」合集第 14 站,讲 AI 通识。

## 已确认决策(用户拍板)

1. **内容侧重**:通识全景为主线,**融合大模型/LLM 主线**;架构要能**持续加最新内容**。
2. **引擎/工具**:克隆历史站引擎(`world-history`),**保留 AI 发展史时间轴 + 人物图鉴**,**去掉小测(quiz)**。
3. **视觉**:要有创新、明显区别于现有站。出了两轮共 6 个 demo(A 神经脉络 / B Token 流 / C 智能体终端,以及 D 文档手册 / E 对话流 / F 杂志编辑),用户选定 **D 文档手册**,并要求**注意移动端适配**。
4. **规模**:comprehensive 全量(13 模块 / 100 课),且**可扩展**(后续加专用工具讲解等)。
5. **首批互动工具(3 个)**:① Transformer 注意力可视化 ② 神经元/感知机互动 ③ 提示词演练器。均离线、教学示意。
6. **新增章节**:
   - 工程进阶模块:`Prompt → Context → Harness → Loop` 四层工程(向小白讲)。
   - 智能体模块内含 **Agent 工具对比**(Claude Code / Codex / OpenClaw / Hermes 等)。
7. **人物图鉴**:中外关键节点**全覆盖**(图灵、麦卡锡、明斯基、罗森布拉特、Hinton/LeCun/Bengio、李飞飞、吴恩达、Hassabis、Vaswani、Sutskever、Altman、黄仁勋、Goodfellow;中方:张钹、周志华、陆奇、梁文锋、杨植麟、汤晓鸥等)。

## 落地结果

- 命名空间 `WHS → AIX`;localStorage `aix.progress.v1`;品牌色 `#4f46e5`。
- 13 模块 / 100 课 / 73 术语 / 21 人物 / 17 时间节点 / 大模型速查 + 中外对照。
- 风格 D 文档手册:三栏(模块树 248px + 正文 + 大纲 212px),浅/深主题,≤860px 折叠为单栏 + 抽屉式侧栏(汉堡按钮 + 遮罩)。
- 互动实验室三件套全部可操作;预留 `#/lab/<名>` 扩展位。
- 内容由 `tools/`(gen.js + lessons.js + emit.js)单一数据源生成,带校验,便于持续增补。

## 刻意的取舍 / 注意

- **去掉了小测**:课程只保留 一句话/讲透/别踩坑/接着读 四段。
- **不调用真实模型**:全站离线;实验室演示是规则驱动的教学示意,已在界面显著标注。
- **口径截至 2026 年初**:模型名称、参数、排名等会过时,正文一律强调记趋势与原理。
- 人物在正文中用纯文本引用(`fg`),不做术语弹卡,避免与术语体系混淆。
