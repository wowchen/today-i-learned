# 投资理财学习站 · Personal Finance

用大白话把**投资理财**讲透的科普站——从一张工资条,到资产配置、股债基金、房产保险、退休税务、风险管理。
面向工资党与家庭理财,**不是考试站**:没有真题与错题本,每课只有轻量"想一想"帮你停下来思考。

纯静态、零依赖、离线可用;术语点查、4 个理财计算器、深/浅色切换。是「今日得到 / Today I Learned」合集的第 10 个站。

> **重要:本站不荐股、不带货、不焦虑,不构成任何投资建议。** 案例数据均为假设;税率、个税专项扣除、养老政策等以现行法规为准。

## 怎么用

- 顶部导航:**课时 / 术语 / 计算器 / 收藏 / 搜索 / 设置**,右上角可切换深/浅色。
- 跟着首页模块目录走(入门→核心→进阶→实战),每课约 5 分钟。
- 每课五段:① 一句话 · ② 讲透 · ③ 别踩坑 · ④ 想一想 · ⑤ 接着读;正文带下划线术语点开即看、可收藏。
- 计算器内置:复利、定投估算、退休缺口、贷款月供。
- 进度/收藏存浏览器本地,设置可导出备份。

## 本地运行

```bash
# 仓库根目录
python3 -m http.server 8012
# 打开 http://localhost:8012/personal-finance/
```

## 内容

11 个模块、70 节五分钟微课:
- 入门:理财通识、复利与时间价值
- 核心:资产配置、股票基础、债券与利率、基金与 ETF
- 进阶:房产与 REITs、保险规划、退休与养老
- 实战:税务基础、风险管理与行为偏差
