# 网规学习站 · 网络规划设计师(软考高级)

用大白话讲透**网络规划设计师**考点的备考站。三科全覆盖:综合知识(选择,含专业英语)、案例分析、论文。
纯静态、零依赖、离线可用;真题练习即时判分、错题本、网规计算器、术语点查。

是「今日得到 / Today I Learned」合集中的一个站,引擎沿用考试站(瑞士编辑型 + 琥珀金配色)。

## 怎么用

- 顶部导航:**课时 / 术语 / 收藏 / 错题 / 计算器 / 搜索 / 设置**
- 跟着首页学习路线走(基础→核心→进阶→实战),也可点模块卡片跳学
- 每课⑤有真题练习,答错自动进错题本;正文带下划线术语点开即看、可收藏
- 计算器内置:IP 子网划分、信道容量(奈氏/香农)、链路可靠度(串/并联)
- 进度/收藏/错题存浏览器本地,设置可导出备份

## 本地运行

```bash
# 仓库根目录
python3 -m http.server 8011
# 打开 http://localhost:8011/network-planning-designer/
```

## 内容

21 个模块、98 节五分钟微课:
- 基础:数据通信、网络体系结构(OSI/TCP-IP)、物理层与传输介质、数据链路层
- 核心:网络层与 IP(子网 CIDR/VLSM)、路由协议、传输层、应用层服务、局域网与交换、广域网与接入、网络互连设备
- 进阶:网络安全、无线与移动网络、网络存储、新一代网络(SDN/NFV)、网络规划与设计、网络运维与管理、标准化与专业英语
- 实战:案例分析专题、论文写作专题

> 考试题量/分值/法律年限等以官方最新通知与现行法律为准。开发约定见 `CLAUDE.md`。
