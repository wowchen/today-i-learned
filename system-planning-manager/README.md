# 系规学习站 · 系统规划与管理师(软考高级)

用大白话讲透**系统规划与管理师**考点的备考站。两科全覆盖:综合知识(选择,含专业英语)、案例分析。**本考不考论文。**
纯静态、零依赖、离线可用;真题练习即时判分、错题本、系规计算器、术语点查。

是「今日得到 / Today I Learned」合集中的一个站,引擎沿用考试站(瑞士编辑型 + 橄榄绿配色)。

## 怎么用

- 顶部导航:**课时 / 术语 / 收藏 / 错题 / 计算器 / 搜索 / 设置**
- 跟着首页学习路线走(基础→核心→进阶→实战),也可点模块卡片跳学
- 每课⑤有真题练习,答错自动进错题本;正文带下划线术语点开即看、可收藏
- 计算器内置:服务可用性(MTBF/MTTR)、投资回收期、服务人力测算
- 进度/收藏/错题存浏览器本地,设置可导出备份

## 本地运行

```bash
# 仓库根目录
python3 -m http.server 8011
# 打开 http://localhost:8011/system-planning-manager/
```

## 内容

18 个模块、94 节五分钟微课:
- 基础:计算机与信息系统 / 网络与通信 / 数据库 / 信息安全
- 核心:IT 服务与 ITSS 体系 / 服务规划设计 / 部署实施 / 运营管理 / 持续改进
- 进阶:监督管理 / 团队人员 / 服务营销 / 项目管理 / 工具新技术 / 标准化知识产权 / 专业英语
- 实战:案例分析专题

> 考试题量/分值/法律年限等以官方最新通知与现行法律为准。开发约定见 `CLAUDE.md`。
