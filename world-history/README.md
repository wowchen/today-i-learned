# 世界历史 · World History

大白话讲透世界历史的科普通识站。从文明曙光到当代，**12 个时代 / 122 节微课**，先建立整体画面感，再谈细节。与本合集中的「[中国历史](../chinese-history/)」站形成「中 / 外」一对。

纯静态 · 零依赖 · 离线可用 · 点开即学。

## 四件工具

- **时间轴** —— 公元前 3500 到今天的时代条带，点段跳课时。
- **人物图鉴** —— 约 75 位世界历史人物，按时代/角色过滤，看生平与关联课时。
- **文明速查** —— 12 个时代/文明一览表：起讫、地域、代表政权、关键转折。
- **中外对照** —— 同一时间线左看世界、右看同期中国，建立东西方坐标。

## 原则

多文明并重、不以单一文明为中心；不戏说、不阴谋论、不焦虑；学术争议处标注通说与异说；涉及中国处与「中国历史」站口径一致；当代政治与在世人物只述事实、不作价值判断。

## 技术

纯 HTML/CSS/JS，零依赖、零构建，Hash SPA 路由，`localStorage` 存进度（前缀 `whs.`）。深/浅色与字号可切换。内容由单一数据源生成器产出，课时 id 与 `WHS.path` 一一对应。详见 [CLAUDE.md](CLAUDE.md)。
