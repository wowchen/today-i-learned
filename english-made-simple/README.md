# English Made Simple · 美语,一看就懂

用大白话**中文**讲透**美式英语**的零基础自学站。一看就懂,点哪听哪,边学边练。

🌐 **在线使用(手机 / 电脑 / 平板都行):** https://wowchen.github.io/english-made-simple/

打开网址即可,**无需注册、无需安装、无需任何配置**。

---

## 这是什么

一个面向**零基础小白**、用通俗中文讲解的**美式英语**学习网站:

- **全领域**:发音、词汇、语法、听力、口语、阅读、写作、文化语用,九大模块共 **97 课**。
- **一看就懂**:每节课都用同一套结构 —— ①一句话是什么 → ②什么时候用 → ③怎么用 → ④中国人易踩的坑 → ⑤30 秒小测。
- **美式为准**:发音、拼写、表达全部 American English。
- **纯静态**:原生 HTML/CSS/JS,零依赖、零构建,双击 `index.html` 也能离线跑。

## 核心功能

| 功能 | 说明 |

| 🌗 **深/浅主题 · 字号调节 · 全文搜索 · 移动端自适应** | |

## 怎么用(学习者)

1. 打开 https://wowchen.github.io/english-made-simple/
2. 首页点 **「继续学习」**,跟着学习路线从第 1 课开始(零基础建议顺着走)。
3. 遇到英文**点一下听发音**;想记的词**点「☆ 收藏」**进生词本。
4. 每课末尾做 **30 秒小测**,做错的自动进错题本。
5. 用 **「快速复习」** 隔三差五把生词本/错题本过一遍。
6. 想跳着学,用右侧便利贴标签或首页目录直达任意模块。

> 💡 每个人的学习进度都存在**自己浏览器**里,各自独立、互不可见。

## 跨设备同步(可选)

想让进度在多台设备间同步(如家里和公司),在站内 **「设置 → GitHub 进度同步」** 里填:

- 一个**自己的 private 仓库**(如 `你的用户名/english-progress`)
- 一个 **fine-grained PAT**(只授权该仓库 `Contents` 读写,建议设 90 天过期)

Token 只存在你自己浏览器的 localStorage,**不会进入任何代码或公开仓库**。不配置则进度仅存本机,完全不影响学习。

## 本地运行

纯静态,无需任何依赖:

```bash
# 方式一:直接双击 index.html(file:// 即可)
# 方式二:起个本地服务器
python3 -m http.server
# 然后浏览器打开 http://localhost:8000
```

## 技术说明

- 原生 HTML / CSS / JavaScript,**无框架、无构建步骤**。
- 内容即数据:每课是一条结构化数据(`content/**/*.js`),由统一渲染器吐成页面;`file://` 下用 `<script>` 注册避开 CORS。
- 发音用浏览器 **Web Speech API(SpeechSynthesis)**,优先本地 `en-US` 嗓音以保证离线。
- 想新增课程或了解架构,见 [`CLAUDE.md`](./CLAUDE.md)(开发规范)与 [`需求与讨论.md`](./需求与讨论.md)(设计决策 ADR)。
