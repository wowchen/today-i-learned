> 信息系统项目管理师（高项 / 软考高级）备考学习站。纯静态、零依赖、离线可用。

## 项目简介

为备考「信息系统项目管理师」（软考高级资格）打造的学习站。把厚厚的考纲拆成 **134 节 5 分钟微课**，每课白话先行、考纲导向，配真题练习和易混辨析，覆盖综合知识、案例分析、论文三科。

- **纯前端零依赖**：纯 HTML/CSS/JS，无 npm、无构建、无后端，`file://` 双击即可打开
- **离线可用**：所有内容打包在仓库内，断网也能学
- **进度本地保存**：学习进度、收藏术语、错题本存在浏览器 localStorage

## 内容体系（18 模块 / 134 课时）

| 板块 | 模块 | 课时 |
|------|------|------|
| **项目管理核心** | 学习导览、项目管理框架、范围、进度、成本、质量、风险、资源与沟通、采购、整合与变更 | 64 |
| **技术与基础** | 信息化与信息系统、系统工程、软件工程、IT 基础设施、项目集与项目组合 | 46 |
| **法规与实战** | 法律法规与标准、案例分析专题、论文写作专题 | 32 |

每节课采用**六段式模板**：① 概念（白话+类比）② 考纲定位（出题频率/分值）③ 核心要点（机制/图解）④ 易混辨析（真实易错点）⑤ 真题练习（2-3 题带解析）⑥ 关联。

## 功能特性

- **术语弹窗**：课文中术语可点击，弹出定义、英文、生活类比，可一键收藏
- **真题练习**：选择/填空题即时判分，答错自动进错题本
- **错题本 / 我的术语本**：集中复习薄弱点
- **案例计算器**（`#/calc`）：挣值管理 EVM、三点估算 PERT、沟通渠道数，输入即算
- **全文搜索**：跨课时和术语检索
- **学习路径**：按"先建框架、再填细节、最后实战"的认知顺序排课
- **深 / 浅主题、字号调节、进度导出**
- **移动端适配**：手机、平板、桌面自适应

## 技术架构

- 纯 HTML/CSS/JS，零依赖、零构建步骤
- Hash SPA 路由（`#/l/module/lesson`）
- Content-as-Data：每节课是一个 JS 对象，通过 `ISPM.registerLesson()` 注册
- localStorage 持久化，全局命名空间 `window.ISPM`
- 视觉风格：瑞士国际主义编辑排版（黑白底 + 单一橙强调 + 强网格）

## 本地运行

无需任何环境，两种方式任选：

```bash
# 方式一：直接双击 index.html 用浏览器打开

# 方式二：起一个本地静态服务器（推荐，避免个别浏览器的 file:// 限制）
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 添加 / 修改内容

1. 在 `content/<module>/` 下创建或编辑 `<NN>-<slug>.js`，按六段式模板编写
2. 在 `index.html` 底部添加 `<script src="...">` 引用
3. 确认 `content/modules.js` 的学习路径 `ISPM.path` 包含该课时 ID
4. 新术语加到 `content/terms.js`
5. 校验语法：`node --check content/<module>/<file>.js`

详细规范见 [CLAUDE.md](CLAUDE.md)。
