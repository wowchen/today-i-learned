# 中国历史学习站 · Chinese History

用大白话讲透**中国五千年历史**的科普站——从神话开篇(盘古、女娲、三皇五帝),到夏商周秦汉、隋唐宋元明清,再到民国与新中国。
**不戏说、不阴谋论、不焦虑**,给非历史专业的普通读者一个整体而清晰的中国历史画面感。

纯静态、零依赖、离线可用;术语点查、时间轴、人物图鉴、朝代速查、皇帝年号换算器、深/浅色切换。是「今日得到 / Today I Learned」合集的第 11 个站。

## 怎么用

- 顶部导航:**课时 / 时间轴 / 图鉴 / 速查 / 年号 / 术语 / 收藏 / 搜索 / 设置**,右上角可切换深/浅色。
- 跟着首页 14 块朝代瓦片走(神话→夏商周→…→改革开放),每课约 5 分钟。
- 每课五段:① 一句话 · ② 讲透 · ③ 别踩坑 · ④ 想一想 · ⑤ 接着读;正文带下划线术语点开即看、可收藏。
- 工具内置:**5000 年时间轴**(全局条带 + 朝代内事件流)、**人物图鉴**(约 96 位)、**朝代速查表**(14 朝代一览)、**年号换算器**(约 280 个主要年号)。
- 进度/收藏存浏览器本地,设置可导出备份。

## 本地运行

```bash
# 仓库根目录
python3 -m http.server 8013
# 打开 http://localhost:8013/chinese-history/
```
