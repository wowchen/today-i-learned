# 十五五规划学习站 · 需求与讨论(本站宪法)

> 这是本站的"宪法":需求、关键决策(ADR)、体验原则。改内容前先读它,不要违背。

## 一句话定位

用**大白话中文**把《国民经济和社会发展第十五个五年规划纲要》(2026–2030)讲透的零基础自学站。
科普为底、产业/决策视角收尾。纯静态、零依赖、离线可用,GitHub Pages 托管。

## 受众与口吻(已与站长确认)

- 受众:对"十五五"好奇但没系统读过文件的普通人。
- 口吻:**零基础大白话科普为主**,每课末尾给一条"对你/对行业意味着什么"的落点(产业/决策视角)。
- 不堆文件原话、不掉书袋;先把概念和逻辑讲清,术语点开即查。

## 内容来源

- 主源:《中华人民共和国国民经济和社会发展第十五个五年规划纲要》(2026-03 十四届全国人大四次会议表决通过,覆盖 2026-01-01 至 2030-12-31)。ISBN 9787010280912。
- 辅源:2026 年《政府工作报告》;中国政府网(gov.cn)相关公报与解读;百度百科纲要词条(结构与指标核对)。
- **准确性原则**:数值/表述以《纲要》正式文本为准。科普稿中的具体数字若来自二手报道,必须标注"概数,以原文为准"(见 `overview/04-index.js` 的处理范式)。

## 结构(定稿)

- 纲要共 **18 篇 / 62 章**。本站设 **19 个模块**:模块 0「导览」+ 第一至十八篇各一模块。
- 学习路线共 **63 课**(见 `content/modules.js` 的 registerPath)。未编写的课显示"待编写",写好即自动激活。
- 五段课模板(引擎固定渲染,见 `assets/js/views-lesson.js`):
  ① 一句话是什么 · ② 为什么重要 · ③ 怎么运作 · ④ 常见误解与易混 · ⑤ 关联与延伸。

## 关键决策(ADR)

1. **复用电网站引擎**:整套 SPA 引擎从 `power-grid-fundamentals` 复制,内容即数据(`<script>` 注册,file:// 可跑,零构建)。**不改 `assets/` 下任何引擎代码**。
2. **风格定稿 = B 智库白皮书**:白底留白 + 理性蓝(#1B4F8A)+ 青绿(#0E9594),无衬线。三套选型稿保留在 `design/`,不要删。设计令牌集中在 `assets/css/main.css` 顶部 `:root`;B 专属组件微调在文件末"风格 B 覆盖"段。默认浅色主题(`data-theme="light"`),另留深蓝夜读主题。
3. **全局命名空间 = `FYP`**:复制引擎后已将 `PGF` 全量改名为 `FYP`,避免与电网站语义混淆。内容文件一律调用 `FYP.registerLesson` 等。
4. **localStorage 键加前缀 `ffp.`**:本站与电网站等子站同源(`wowchen.github.io/today-i-learned/`)部署,进度键 `ffp.progress.v1`、同步配置键 `ffp.sync.v1`,避免跨站串数据。
5. **数字标注纪律**:涉及量化指标必须可回溯到《纲要》;二手概数显式标注,宁可保守不可编造。
6. **政治表述纪律**:本站是个人科普站,做"通俗解读",不做权威发布;遇敏感/重大表述以原文为准,不擅自演绎。

## 体验原则

- 一节课几分钟读完,能独立成篇。
- 术语**点哪查哪**(`<gd data-term="id">`),解释给类比。
- 不用 emoji;图标一律线性 SVG 或字形符号。
- 手机/电脑/平板都耐看;进度只存本机,跨设备同步可选(设置里配 GitHub PAT)。
