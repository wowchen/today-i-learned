党政红金 · 政策读本

暖白纸底 + 党政红 + 描金 + 宋体大标题 + 红头横线/印章气质。像一本官方学习读本，庄重权威。
 B

智库白皮书 · 数据留白

纯白大留白 + 理性蓝 + 青绿 + 无衬线 + 大号数字块。像一份研究院白皮书，现代克制、图表导向。
 C

编年蓝图 · 战略仪表

深蓝底 + 荧光青 + 描金 + 等宽标注 + 五年时间轴 + 仪表盘。像一张国家发展战略图，科技理性。
