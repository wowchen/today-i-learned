# 医疗业务与信息化学习站 · Healthcare Business & IT

「今日得到」合集中的第 15 个学习站。用大白话讲透医疗业务与信息化:从医院业务流程、医保支付、医疗质量安全,到 HIS、EMR、LIS、PACS、数据标准、互联互通、安全合规、互联网医疗与智慧医院。

- **纯静态、零依赖、离线可用**:打开 `index.html` 即可学,无需服务器、不联网。
- **13 模块 / 100 节微课**:一课一个点,约 5 分钟读完。
- **工具**:医疗改革与信息化时间轴、角色/机构图鉴、医疗缩略语速查、传统流程 vs 数字化流程对照。
- **互动实验室**:① 就诊流程模拟器 ② DRG/DIP 教学示意器 ③ 医疗缩略语速配。
- **视觉**:A「临床病历」配色 + C「互联互通蓝图」设计结构,浅/深色主题,移动端抽屉式侧栏。

## 使用

直接用浏览器打开本目录的 `index.html`;或在合集根目录起一个静态服务器后访问 `/healthcare-business-it/`。学习进度只存在本机浏览器(localStorage `hit.progress.v1`),可在「设置」里导出备份。
