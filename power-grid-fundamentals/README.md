# 电网通识 · Power Grid 101

用大白话**中文**讲透**中国电力/电网行业**的零基础自学站。一看就懂,术语点哪查哪。

**在线使用(手机 / 电脑 / 平板都行):** https://wowchen.github.io/20260611-power-grid-fundamentals/

打开网址即可,**无需注册、无需安装、无需任何配置**。

## 这是什么

一个面向**零基础**(非电力专业出身)、用通俗中文讲解的电网行业通识网站:

- **全领域**:能源全景、电的物理常识、发输变配用、调度运行、电网格局与电改、监管与国网组织、电力市场、新能源、储能、营销电价、双碳、数字化、能源金融、气水氢,**17 个模块共 138 课,已全部上线**。
- **一看就懂**:每课同一套结构——①一句话是什么 → ②为什么重要 → ③怎么运作 → ④常见误解 → ⑤关联延伸,5 分钟一课。
- **不考试**:没有任何小测判分,纯阅读 + 术语回顾。
- **纯静态**:原生 HTML/CSS/JS,零依赖、零构建,双击 `index.html` 也能离线跑。

## 核心功能

| 功能 | 说明 |

| 我的术语本 | 卡片里点「收藏」,攒成自己的复习清单 |

| 连续学习记录 · 深/浅主题 · 字号调节 · 全文搜索 · 移动端自适应 | |

## 怎么用(学习者)

1. 打开网站(或本地双击 `index.html`)。
2. 首页点 **「继续学习」**,跟着路线从第 1 课开始,138 课全部可读。
3. 遇到看不懂的术语**点一下**看解释;想记住的点**「收藏」**进术语本。
4. 隔三差五用 **「翻牌复习」** 把术语本过一遍。
5. 想跳着学,用右侧图签或首页目录直达任意模块。

> 每个人的学习进度都存在**自己浏览器**里,各自独立、互不可见。

## 跨设备同步(可选)

想让进度在多台设备间同步,在站内 **「设置 → GitHub 进度同步」** 里填:

- 一个**自己的 private 仓库**(如 `你的用户名/grid-progress`)
- 一个 **fine-grained PAT**(只授权该仓库 `Contents` 读写,建议设 90 天过期)

Token 只存在你自己浏览器的 localStorage,**不会进入任何代码或公开仓库**。不配置则进度仅存本机,完全不影响学习。

## 本地运行

纯静态,无需任何依赖:

```bash
# 方式一:直接双击 index.html(file:// 即可)
# 方式二:起个本地服务器
python3 -m http.server
# 然后浏览器打开 http://localhost:8000
```

## 技术说明

- 原生 HTML / CSS / JavaScript,**无框架、无构建步骤**。
- 内容即数据:每课是一条结构化数据(`content/**/*.js`),由统一渲染器吐成页面;`file://` 下用 `<script>` 注册避开 CORS。
- 视觉为**工程蓝图**风:深主题 = 蓝晒图纸,浅主题 = 白图纸;全站不用 emoji,图标一律线性。
- 想新增课程或了解架构,见 [`CLAUDE.md`](./CLAUDE.md)(开发与内容规范)与 [`需求与讨论.md`](./需求与讨论.md)(设计决策 ADR)。
