# 系分学习站 · 系统分析师(软考高级)

用大白话讲透**系统分析师**考点的备考站。三科全覆盖:综合知识(选择)、案例分析、论文。
纯静态、零依赖、离线可用;真题练习即时判分、错题本、系分计算器、术语点查。

是「今日得到 / Today I Learned」合集中的一个站,引擎沿用考试站(瑞士编辑型 + 靛紫配色)。

## 怎么用

- 顶部导航:**课时 / 术语 / 收藏 / 错题 / 计算器 / 搜索 / 设置**
- 跟着首页学习路线走(基础→核心→进阶→实战),也可点模块卡片跳学
- 每课⑤有真题练习,答错自动进错题本;正文带下划线术语点开即看、可收藏
- 计算器内置:决策树期望值、PERT 三点估算、系统可靠度(串/并联)
- 进度/收藏/错题存浏览器本地,设置可导出备份

## 本地运行

```bash
# 仓库根目录
python3 -m http.server 8011
# 打开 http://localhost:8011/system-analyst/
```

## 内容

18 个模块、122 节五分钟微课:
- 基础:计算机系统 / 操作系统 / 数据库 / 网络与信息安全
- 核心:软件工程 / 需求工程 / 结构化与面向对象分析设计 / 系统架构
- 进阶:系统规划与可行性 / 可靠性与安全性 / 测试转换维护 / 信息系统综合 / 新技术 / 数学与经济管理 / 标准化
- 实战:案例分析专题 / 论文写作专题

> 考试题量/分值/法律年限等以官方最新通知与现行法律为准。开发约定见 `CLAUDE.md`。
